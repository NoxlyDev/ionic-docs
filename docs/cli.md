AETHERION SMP is a competitive **Lifesteal SMP** server with custom features, unique gameplay, and an active community.  
This guide explains **how to join** the server from both Java Edition and Bedrock Edition.

---

## Server Information

**Server Name:** AETHERION SMP  
**IP Address:** `ae.noxlydev.xyz`  
**Port:** `20001`  
**Version:** Java/Bedrock Crossplay Support (Via Geyser/Floodgate)  
**Mode:** Lifesteal SMP

---

## Requirements

- A supported Minecraft version (Java or Bedrock)
- Stable internet connection
- Make sure your game is updated
- No VPN/Proxy that blocks connections

---

## How to Join (Java Edition)

1. Launch **Minecraft Java Edition**  
2. Go to **Multiplayer**
3. Click **Add Server**
4. Enter the following:
   - **Server Name:** AETHERION SMP  
   - **Server Address:** `ae.noxlydev.xyz:20001`
5. Click **Done**
6. Select the server and click **Join**

---

## How to Join (Bedrock Edition)

1. Open **Minecraft Bedrock**
2. Go to **Play**
3. Select **Servers**
4. Scroll down and choose **Add Server**
5. Fill the fields:
   - **Server Name:** AETHERION SMP  
   - **Server Address:** `ae.noxlydev.xyz`  
   - **Port:** `20001`
6. Press **Save**
7. Tap **Join Server**

---

## Troubleshooting

If you cannot join the server, try the following:

- Make sure you typed the IP and port correctly
- Restart your Minecraft client
- Check your internet connection
- Disable VPN or proxy apps
- If using Bedrock, make sure you entered the correct port (`20001`)
- Restart your device if needed
